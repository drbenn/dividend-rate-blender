Enter an active ticker from the NYSE and a dollar amount to invest. Investing into multiple dividend stocks with different dividend rates, you can see how much to invest in which stock to hit your desired dividend yield.


This tool uses the free version of FMP API, only 250 calls are allowed per day.

If the FMP API calls are expensed or currently offline, four dummy variables listed below with different dividend yield rates have been created for demonstration purposes;

  • DIRT - 1%
  • ROCK - 3%
  • SAND - 5%
  • LAVA - 9%
